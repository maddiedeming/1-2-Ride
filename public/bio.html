    1-2 Ride

    A simple application comparing Lyft and Uber. Our project is designed to give you the capability of searching for available Uber and Lyft rides in a side by side comparison. Will it effect the user's decision?

    The following technologies were used in this application:
      Bootstrap v.4 Jquery Geolocation services Firebase Database Uber API Lyft API Chart.js
